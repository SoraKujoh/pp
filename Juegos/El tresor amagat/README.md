# El tresor amagat

Juego sencillo donde controlas un personaje con las flechas del teclado para encontrar un tesoro que aparece en una posición aleatoria.

Cómo jugar:
- Abrir `Juegos/El tresor amagat/index.html` en tu navegador (doble clic o arrastrar al navegador).
- Moverte con las flechas del teclado.
- Cuando alcances el tesoro aparece una pantalla de victoria y puedes reiniciar con el botón.

Notas técnicas:
- Juego implementado con `canvas` y sprites dibujados como píxeles (sin imágenes externas).
- Tamaño del mapa: 20x15 tiles, tile = 16 px.
